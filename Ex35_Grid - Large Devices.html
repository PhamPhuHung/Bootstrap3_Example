Bootstrap Grid Example: Large Devices

Extra Small Small Medium Large
Class prefix .col-xs .col-sm .col-md .col-lg
Screen width <768px >=768px >=992x >=1200px

In the previous chapter, we presented a grid example with classes for small & medium devices. We used 2 divs (columns) and we gave them a 25%/75% split on small devices, andn a 50%/50% split on medium devices:

< div class ="col-sm-3 col-md-6"> ... < /div >
< div class ="col-sm-9 col-md-6"> ... < /div >

But on large devices, the design may be better as a 33%/66% split

Tip: Large devices are defined as having a screen width from 1200 and above

For medium devices we will use the .col-lg-* classes

Now we will add the column widths for large devices:

< div class ="col-sm-3 col-md-6 col-lg-4"> ... < /div >
< div class ="col-sm-9 col-md-6 col-lg-8"> ... < /div >

Now Bootstrap is going to say "at the small size", look at classes with -sm in them and use those. At the medium size, look at classes with -md in them and use those. At the large size, look at classes with the word -lg in them and use those"

The following example will result in a 25%/75% split on small devices, a 50%/50% split on medium devices, and a 33%/66% split on large devices. On extra small devices, it will automatically stack (100%)
