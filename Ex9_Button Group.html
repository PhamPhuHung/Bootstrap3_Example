Button Group

Bootstrap allows you to group a series of buttons together (on a single line) in a button group:

Use a <div> element with class .btn-group to create a button group



Tip: Instead of applying button sizes to every button in a group, use class .btn-group-lg|sm|xs to size all the buttons in the group
