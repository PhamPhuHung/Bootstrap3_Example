Bootstrap provides an easy way to create predefined alert messages:

Basic Alert
Success! This alert box indicates a success full or positive action
Info! This alert box indicates a neutral informative change or action
Warning! This alert box indicates a warning that might need attention
Danger! This alert box indicates a dangferous or potentially negative action

Close the alert

To close the alert message, add a .alert-dismissible class to the alert container. Then add class="close" and data-dismiss="alert" to a link or a button element to make the x show at the end of the div and link it with the alert class (when you click on this the alert box will disapear)

To help improve accessibility for people using screen readers, you shoudl include the aria-label="close" attribute, when creating a close button

× is an HTM entity that is the preferred icon for close buttons, rather than the letter "x"
