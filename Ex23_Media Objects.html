Media Objects

Bootstrap provides an easy way to align media objects (like images or videos) to the left or to the right of some content. This can used to display blog coments tweets and so on:

Basic Media Object

Use a <div> element with the .media class to create a container for media objects

Use the .media-left class to align the media object (image) to the left, or the .media-right class to align it to the right

Text that should appear next to the image, is place inside a container with class="media-body"

Additionally, you can use .media-heading for headings
