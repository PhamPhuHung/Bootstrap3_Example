Bootstra's Default Setting

Form controls automatically receive some global styling with Bootstrap:

All texttual <input>, <textarea>, and <select> elements with class .form-control have a width of 100%

Bootstrap Form Layouts

Bootstrap provices 3 types of form layouts:

  • Vertical form (default)
  • Horizontal form (default)
  • Inline form (default)

Standard rules for all 3 form layouts:

  • Wrap labels and form controls in <div class="form-group"> (needed for optimum spacing)
  • Add class .form-control to all textual <input>, <textarea>, and <select> elements<
  • Vertical form (default)

Tip: You can hide the labels for all devices, except screen readers, by using the class

Bootstrap vertical Form (default)

Bootstrap Inline Form

In an inlien form, all of the elements are inlien, left-aligned, and the labeles are alongside.

Note: This only applies to forms within viewports that are atleast 67px wide

Additional rule for an inlie form: add class .form-inline to the <form> element

Bootstrap Horizontal Form

A horizontal form means that the labels are aigned next to the input field (horizontal) on large and medium screen. On small screens (767px and below), it will transform to a vertical form (abels are placed on top of each input)

Additional rules for a horizontal form:

  • Add class .form-horizontal to the <form> element
  • Add class .control-label to all <label> element

Tip: Use Bootstrap's predefined grid classes to align labels and groups of form controls in a horizontal layout.
